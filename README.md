# Writers Paradise

A Laravel MVC-based web application for managing writers, books, orders, and cart functionality.

## Features
- Role-based authentication
- CRUD operations for writers and books
- Cart management (add, remove, increment, decrement)
- Order processing and stock validation
- Relational database with foreign key constraints
- Session-based state management
- YouTube API integration for dynamic media embedding

## Tech Stack
- Laravel (MVC Architecture)
- PHP
- MySQL
- Blade Templates
- RESTful Routing

## Key Backend Concepts Implemented
- Database schema design with migrations
- Controller-based business logic
- Route protection & middleware
- Relational data modelling
